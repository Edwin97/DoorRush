# Installation
## CocoaPods
To install the dependencies, simply run the following line:

```ruby
pod install
```



## Compatibility

This project is written in Swift 5.0 and requires Xcode 11.5 or newer to build and run.

DoorRush iOS app is compatible with iOS 13.5+.
